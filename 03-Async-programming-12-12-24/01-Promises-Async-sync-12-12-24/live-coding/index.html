Synchron Code und Promises

Synchron Code

Synchroner Code wird sequenziell ausgeführt, Jede Operation muss abgeschlossen sein, bevor die nächste beginnt, Blockierende Operationen sind typischerweise synchron Das System wartet, bis jede Operation nacheinander abgeschlossen ist

Promises oder Asynchroner Code

Asynchroner Code beginnt etwas zu tun, Aber er lässt den Rest des Programms nicht warten, weil Er ist nicht blockierend, Er kann zur nächsten Operation übergehen, während er wartet
